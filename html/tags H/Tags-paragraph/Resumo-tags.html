📚 Tags de Formatação em HTML

As tags de formatação do HTML permitem organizar e destacar o conteúdo de uma página, tornando-o mais agradável para o usuário e mais fácil de interpretar pelos navegadores. Conheça algumas das principais tags utilizadas no desenvolvimento web.

🔤 Tag <b> - Texto em Negrito

A tag <b> deixa um texto em negrito, alterando apenas sua aparência visual.

Exemplo:

<b>Este texto está em negrito.</b>

Resultado:

Este texto está em negrito.

💪 Tag <strong> - Texto Importante

A tag <strong> também exibe o texto em negrito, porém possui significado semântico, indicando que aquele conteúdo é importante.

Exemplo:

<strong>Este texto possui grande importância.</strong>

Este texto possui grande importância.

↩️ Tag <br> - Quebra de Linha

A tag <br> é utilizada para inserir uma quebra de linha sem criar um novo parágrafo.

Exemplo:

Primeira linha.<br>
Segunda linha.

Primeira linha.
Segunda linha.

✨ Tag <em> - Ênfase no Texto

A tag <em> destaca um texto com ênfase, normalmente sendo exibido em itálico.

Exemplo:

<em>Este texto está em destaque.</em>

Este texto está em destaque.

➖ Tag <hr> - Linha Horizontal

A tag <hr> cria uma linha horizontal utilizada para separar seções do conteúdo.

Exemplo:

Seção 1
<hr>
Seção 2

Seção 1

Seção 2

✍️ Tag <i> - Texto em Itálico

A tag <i> altera apenas a aparência do texto, exibindo-o em itálico.

Exemplo:

<i>Texto em itálico.</i>

Texto em itálico.

🖍️ Tag <mark> - Texto Destacado

A tag <mark> destaca um trecho do texto, semelhante ao efeito de um marca-texto.

Exemplo:

<mark>Este conteúdo merece atenção.</mark>

Este conteúdo merece atenção.

🔍 Tag <small> - Texto Pequeno

A tag <small> reduz o tamanho da fonte, sendo bastante utilizada para observações, notas e direitos autorais.

Exemplo:

<small>Informações complementares.</small>

Informações complementares.

📄 Tag <p> - Parágrafo

A tag <p> define um parágrafo, organizando melhor o conteúdo da página.

Exemplo:

<p>Primeiro parágrafo.</p>

<p>Segundo parágrafo.</p>

Este é o primeiro parágrafo.

Este é o segundo parágrafo.

📋 Resumo das Tags

Tag Função
<b> Aplica negrito visual.
<strong> Destaca um texto importante.
<br> Insere uma quebra de linha.
<em> Aplica ênfase ao texto.
<hr> Cria uma linha horizontal.
<i> Aplica itálico visual.
<mark> Destaca um trecho do texto.
<small> Exibe texto em tamanho reduzido.
<p> Cria um parágrafo.

🚀 Conclusão

Utilizar corretamente as tags HTML melhora a organização, a legibilidade e a acessibilidade de uma página. Além disso, tags semânticas como <strong> e <em> ajudam mecanismos de busca e leitores de tela a compreenderem melhor o significado do conteúdo.
